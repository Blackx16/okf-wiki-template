# The Open Knowledge Format Specification

**Source:** `raw/articles/what-is-okf.md`

## Summary
A definition of the OKF directory layouts, provenance structures, and metadata conventions for structuring notes.

## Key Claims
- Decouples raw source files from synthesized metadata.
- Declares relationship links explicitly in YAML using the `related:` array.
- Optimizes cognitive indexing for autonomous AI systems.

## Connections
- Implements the [[open-knowledge-format]] concept.
- Maintained by the [[open-knowledge-foundation]].
